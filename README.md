# 数据密集型应用设计-中文翻译
翻译初衷：本书其实有中译本（[中国电力出版社-赵军平等译]()），也有网译版（[[冯若航译]](https://github.com/Vonng/ddia)），他们都翻译的很好，我也是一开始看的他们的译本，在此很感谢他们，有了中文译本能让我们读起来省时省力。但之所以自己翻译一遍，一个是为了提升自己的英语能力，再有针对这本书，我只想读符合我个人习惯的翻译版本，虽然这对我来说不是一件容易的事，但希望结果是好的。
